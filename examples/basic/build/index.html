You do this by creating that render this plot:

class Lineplot extends PlotlyComponent {
  plot (gd) {
    Plotly.plot(gd, {
      data: [{
        y: [1, 2, 3]
      }],
      layout: {
        margin: {t: 40, r: 30, b: 30, l: 30}
      }
    });
  }
}

The plotly compoent just uses the loader, which is also very simple to use:

render () {

  </Loader>
